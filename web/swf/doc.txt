cut.swf - плеер для обрезки

передаваемые flashvars параметры: 
	vid - id видео на youtube (параметр "v" в url)

При изменении пользователем границ видео, вызываеся JavaScript функция setLimit(begin, end), куда передаются начало и конец соответсвенно. begin и end - не целочисленные. id видео и его границы нужно сохранить в базе сайта.

player.swf - плеер для проигрывания

передаваемые flashvars парамеры:
	vid - id видео на youtube (параметр "v" в url)
	begin - начало видео
	end - конец видео

логотип для плеера должен находиться по адресу http://mmatica.ru/logo.png
